{# templates/macros/ordenacion.html - Macro reutilizable para encabezados ordenables Este macro genera un clicable que permite ordenar la tabla por esa columna. Al hacer clic, alterna entre ascendente y descendente. Conserva todos los parámetros GET actuales (búsqueda, filtros, página) y solo cambia sort/order. Para usar en cualquier plantilla: {% from 'macros/ordenacion.html' import th_ordenable %} {{ th_ordenable('nombre', 'Nombre', sort_actual, order_actual) }} #} {% macro th_ordenable(clave, etiqueta, sort_actual, order_actual) %} {# Genera un encabezado de tabla ordenable con indicador visual.
